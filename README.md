## 📖 About the Book

**Python in Plain English** is a beginner-friendly Python programming book designed to teach programming through simple explanations, practical examples, and hands-on projects.

Rather than memorizing syntax, readers build real applications while learning the fundamentals of Python step by step.

Whether you're a complete beginner, a student, or someone returning to programming after a break, this book provides a structured learning path with clear explanations and practical exercises.

---

## 🚀 What You'll Learn

- Printing and Console Output
- Variables and Data Types
- User Input
- String Manipulation
- Mathematical Operations
- Conditional Statements
- Loops
- Functions
- Lists and Dictionaries
- Debugging
- Object-Oriented Programming (OOP)
